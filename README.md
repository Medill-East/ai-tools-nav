# AI 工具对比导航

一个帮助用户对比和选择 AI 工具的交互式网页应用。

## 功能特点

- **对比模式**：选择 2-4 个工具进行并排对比
- **详细分析**：每个工具的优劣势和适用人群
- **分类筛选**：按类型快速找到工具
- **收藏功能**：保存感兴趣的工具

## 在线访问

https://medill-east.github.io/ai-tools-nav

## 本地使用

直接在浏览器中打开 `index.html` 即可。
